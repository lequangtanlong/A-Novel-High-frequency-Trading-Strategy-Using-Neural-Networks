A Novel High-frequency Trading Strategy Using Neural Networks
====================================================================

This is the official code repository of the paper **A Novel High-frequency Trading Strategy Using Neural Networks**.

Requirements
------------
You should install indispensable packages and libraries:
   ```sh
    pip install jupyterlab
    pip install matplotlib
    pip install numba==0.56.4
    pip install pandas
    pip install pytorch-lightning==1.2.2
    pip install torchmetrics
    pip install torchsde
    pip install torchvision
   ```

You should also install the backtesting framework using ``pip``:

   ```sh
   pip install hftbacktest
   ```
